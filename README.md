# ASCII Motion Studio

A browser-based creative tool that transforms images into animated ASCII art.  
No installation. No sign-up. Just open and create.

🔗 **https://soumyachoubey.github.io/ascii-motion-studio/**

---

## What it does

Upload any image and watch it become animated ASCII characters in real time.  
Every slider change updates the output instantly.

## Features

- **WebGL rendering** — handles 10,000+ particles at 60fps
- **12 animation modes** — Wave, Vortex, Rain, Explode, Gravity, Glitch and more
- **10 character sets** — Standard ASCII, Blocks, Matrix, Braille, Binary and more
- **9 color themes** — with original image color support
- **Zoom & pan** — on both the source image and ASCII output
- **GIF export** — records and downloads animated GIFs, works everywhere
- **Live camera** — freeze a camera frame and convert it (open file directly in browser)
- **Fully offline** — single HTML file, no server, no dependencies

## How to use

1. Open the live app link above
2. Upload an image (or drag and drop)
3. Choose an animation mode from the sidebar
4. Adjust density, contrast, color and speed in real time
5. Hit **⏺ RECORD** to export an animated GIF

## Run locally

No build step needed. Just download the file and open it:
```
1. Download ascii-motion-studio-v2.html
2. Double-click it to open in your browser
```

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play / Pause animation |
| `Enter` | Re-convert image |
| `R` | Randomize everything |
| `P` | Export PNG |
| `I` | Toggle invert |
| `Scroll` | Zoom in/out |

## Built with

- Vanilla JavaScript
- WebGL2 (with Canvas 2D fallback)
- Pure-JS GIF encoder (no external libraries)
